# Calculadora de Formulador Arterial de Embalsamamiento
 
Permite a los usuarios determinar la cantidad exacta de químico arterial y agua necesaria para lograr una concentración objetivo en la solución final.

## Funcionalidades

- **Cálculo de diluciones**: Utiliza la fórmula de dilución C1·V1 = C2·V2 para calcular las cantidades necesarias.
- **Presets de concentración**: Botones rápidos para concentraciones comunes (1.5%, 2%, 2.5%).
- **Selección de químicos**: Lista desplegable con químicos arteriales predefinidos.
- **Validación de entrada**: Verifica que los valores sean positivos y lógicos.
- **Interfaz responsive**: Se adapta a diferentes tamaños de pantalla.

## Cómo usar

1. Ingresa el volumen final deseado en litros.
2. Selecciona o ingresa la concentración del químico arterial en botella (%).
3. Ingresa la concentración objetivo en el tanque (%).
4. Haz clic en "Calcular formulación" o presiona Enter en los campos.
5. Revisa los resultados que incluyen:
   - Mililitros de químico arterial
   - Mililitros de agua/diluyente
   - Relación de mezcla
   - Verificación de concentración lograda

## Tecnologías utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos con variables CSS y diseño responsive
- **JavaScript**: Lógica de cálculo y manipulación del DOM

## Notas importantes

- Esta herramienta es un apoyo para la formulación; siempre ajusta según las condiciones específicas del caso.
- Considera factores como edema, deshidratación y respuesta vascular.
- La fórmula base es C1·V1 = C2·V2.
